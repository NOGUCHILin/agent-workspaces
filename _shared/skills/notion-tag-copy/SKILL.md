# Notionタグコピースキル

Notionのテーブルビューで、選択中のセルのタグを下のセルにコピーペーストするスキル。

## 前提条件

- Notionのテーブルビューが開いていること
- コピー元のセル（タグが入っているセル）が選択されていること

## 手順

### 1. 最初のセルにタグを追加（タグがない場合）

1. セルをクリックして選択
2. ダイアログからタグを選択（例：「進捗変更」）
3. Escキーでダイアログを閉じる

### 2. コピー

```
Ctrl+C
```

### 3. 下のセルに移動してペースト

```
↓（下矢印キー）
Ctrl+V
```

### 4. 繰り返し

残りのセルに対して手順3を繰り返す：

```
↓ → Ctrl+V → ↓ → Ctrl+V → ...
```

## 使用例

14件のレコードにタグを追加する場合：

1. 1件目のセルをクリック
2. 「進捗変更」タグを選択
3. Escでダイアログを閉じる
4. Ctrl+C でコピー
5. ↓ → Ctrl+V を13回繰り返す

## ポイント

- Escでダイアログを閉じてからコピーすること
- ペースト後は自動的に次のセルの準備ができる
- ドラッグより確実にタグがコピーされる
