# シフト入力スキル

スプレッドシートの固定休情報をもとに、KING OF TIMEの月別スケジュールを入力する。

## 対象システム

### スプレッドシート（２０２５年シフト）
- URL: https://docs.google.com/spreadsheets/d/17j75NtzOjnDDc8ffr2nQzTmw5tU_nLm0h5pGGvIB0OA/edit
- 固定休の列: AD列
- 対象シート: 入力したい月のシート（例: 2026/2）

### KING OF TIME
- URL: https://s4.ta.kingoftime.jp/admin/
- ログインID: cnwqao1admin
- パスワード: 4669Uprose

## 入力対象

スプレッドシートでシフト（「裏」など）が入っている従業員のみ入力する。

### 除外対象（入力不要）
- 本間久隆
- 野口凜
- 野口博史

## 入力ルール

### 固定休の曜日設定
- **1番目の公休** = 法定休日
- **2番目の公休** = 法定外休日

例: 「火土休み」の場合
- 火曜日 = 法定休日
- 土曜日 = 法定外休日

### 勤務日の設定
- パターン: 従業員ごとの勤務パターン（既存スケジュールから確認）
- 勤務日種別: 平日
- 休暇区分: --

### 公休日の設定
- パターン: --（または既存パターン）
- 勤務日種別: 法定休日 / 法定外休日
- 休暇区分: 公休
- 取得単位: 全日休暇

## 入力手順

### 1. スプレッドシートで固定休を確認
1. スプレッドシートURLにアクセス
2. 対象月のシートを選択
3. AD列（固定休）を確認し、各従業員の固定休をメモ

### 2. KING OF TIMEにログイン
1. KING OF TIME管理画面にアクセス
2. ログインID・パスワードでログイン

### 3. 月別データから従業員を選択
1. ホーム → 月別データ
2. 表示期間を対象月に設定（例: 2026/02）
3. 対象従業員の「タイムカード」ボタンをクリック
4. 「スケジュール編集」ボタンをクリック

### 4. 月別スケジュール登録で一括入力

#### 勤務日の入力
1. コピー元スケジュール:
   - パターン: 従業員の勤務パターンを選択（例: 事務所勤務10-19）
   - 勤務日種別: 「平日」を選択
2. 対象日のチェックボックスをクリック
3. 「登録」をクリック

#### 公休日（法定休日）の入力
1. コピー元スケジュール:
   - パターン: --
   - 勤務日種別: 「法定休日」を選択
2. 「詳細項目設定」を開く:
   - 休暇区分 種別: 「公休」を選択
   - 取得単位: 「全日休暇」を選択
3. 対象日のチェックボックスをクリック
4. 「登録」をクリック

#### 公休日（法定外休日）の入力
1. コピー元スケジュール:
   - 勤務日種別: 「法定外休日」を選択
2. 「詳細項目設定」:
   - 休暇区分 種別: 「公休」
   - 取得単位: 「全日休暇」
3. 対象日のチェックボックスをクリック
4. 「登録」をクリック

### 5. 次の従業員へ
1. ホームに戻る
2. 月別データから次の従業員を選択
3. 手順3-4を繰り返す

## チェックボックス操作のコツ

ページが大きすぎてスナップショットが取れない場合は、JavaScriptで直接操作:

```javascript
// チェックボックスをクリック（日付形式: YYYYMMDD）
const checkbox = document.querySelector('input[value="20260223"][name="schedule_copy"]');
checkbox.click();
```

## 注意事項

- 「不定休」の従業員は固定の公休設定不要
- 3日以上の固定休（例: 水木金休み）は、最初の2日を法定休日・法定外休日として設定
- 既存スケジュールがある日は上書きされる
- ブラウザ操作中にページのスナップショットが大きすぎる場合は `browser_run_code` でJavaScript実行
- 勤務パターンは従業員ごとに異なるので、既存スケジュールから確認すること
