# accelerate-skill 使用書

## 概要

GUIスキル（Playwright MCP）にCLI高速版を追加し、実行時に選択できるようにする。

## 使い方

### 基本コマンド

```
スキルを高速化して: {スキル名}
```

### 具体例

```
ユーザー: yamato-loginスキルを高速化して
ユーザー: morning-kpi-inputにCLI版を追加して
ユーザー: このスキルをAPI化して
```

## 実行フロー

```
1. 対象スキル選択
   ↓
2. SKILL.md分析（サービス、手順、入出力）
   ↓
3. CLI化調査（API/CLI/ライブラリ検索）
   ↓
4. cli.md生成
   ↓
5. スキル再構成（SKILL.md → gui.md + 新SKILL.md）
   ↓
6. 完了報告
```

## 生成されるファイル

| ファイル | 内容 |
|----------|------|
| SKILL.md | 選択肢提示（GUI版 or 高速版） |
| gui.md | 元のGUIワークフロー |
| cli.md | 新規CLI版ワークフロー |
| scripts/ | APIスクリプト（必要時） |

## 高速化後のスキル使用

```
ユーザー: ヤマトにログインして
Claude: GUI版と高速版どちらで実行しますか？
  1. GUI版 - ブラウザ操作（視覚確認可能）
  2. 高速版 - API/スクリプト（速い）

ユーザー: 高速版で
Claude: [cli.mdに従って実行]
```

## 注意事項

- API利用にはキー取得が必要な場合あり
- サービスの利用規約を確認すること
- GUI版は残るので、いつでも戻れる
