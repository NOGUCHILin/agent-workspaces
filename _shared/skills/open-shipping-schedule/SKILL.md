# 発送予定オープンスキル

Notionの発送予定ページを開き、作成日時フィルターを今日の日付に自動設定する。

## 前提条件

- Playwright MCPが有効であること
- Notionにログイン済みであること

## 手順

### 1. Notionの発送予定ページをフルページで開く

```
URL: https://www.notion.so/japanconsulting/ebaadbf255204fddb6098081a1715f9f?v=6bbe9978896c44c7a04ad8476491231c
```

サイドピーク（横からスライドして開く）ではなく、フルページで開く。
サイドピークで開いた場合は「フルページで開く」をクリック。

### 2. 日付フィルターを今日に変更

1. フィルターボタン「作成日時: X月X日」をクリック
2. カレンダーダイアログが開く
3. 今日の日付をクリック
4. Escapeキーでダイアログを閉じる

### 3. 確認

- フィルターが「作成日時: [今日の日付]」になっていることを確認
- リストに今日作成された発送予定が表示されている

## 使用タイミング

- 「発送予定開いて」
- 「発送リスト見せて」
- 「今日の発送を確認」
- 発送業務の開始時

## 備考

- フィルターが既に今日の日付の場合は変更不要
- バックマーケット、ムスビー、返送など全ての媒体が表示される
