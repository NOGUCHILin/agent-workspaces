# シフトコメント確認スキル

スプレッドシートのシフトセルに付いているコメントを確認し、内容を報告する。

## 対象システム

### スプレッドシート（２０２５年シフト）
- URL: https://docs.google.com/spreadsheets/d/17j75NtzOjnDDc8ffr2nQzTmw5tU_nLm0h5pGGvIB0OA/edit
- 対象シート: 確認したい月のシート（例: 2026/2）

## 確認手順

### 1. スプレッドシートを開く
1. スプレッドシートURLにアクセス
2. 対象月のシートタブをクリック

### 2. 従業員を検索
1. Ctrl+F で検索ダイアログを開く
2. 従業員名を入力してEnter
3. 検索結果から従業員の行番号を確認（例: A19）
4. 検索ダイアログを閉じる

### 3. コメントを確認

**重要: セルを左クリックするとコメントが表示される**

#### 方法1: 名前ボックスで移動してクリック
1. 名前ボックス（左上のセル番地表示）をクリック
2. セル番地を入力（例: C19）してEnter
3. そのセルの位置をマウスで左クリック
4. コメントがあれば吹き出しで表示される

#### 方法2: 座標クリック（browser_run_code使用）
```javascript
async (page) => {
  // 名前ボックスでセルに移動
  await page.locator('#t-name-box').click();
  await page.locator('#t-name-box').fill('C19');
  await page.locator('#t-name-box').press('Enter');
  await page.waitForTimeout(500);

  // セルを左クリック（y座標は行に合わせて調整）
  // 19行目の場合: y ≈ 403
  await page.mouse.click(215, 403);
  await page.waitForTimeout(1500);
}
```

### 4. 座標の目安

**列（x座標）:**
- A列: 100
- B列: 185
- C列: 215（02日）
- D列: 245（03日）
- E列: 275（04日）
- 以降、各列 +30px ずつ

**行（y座標）:**
- 行番号 × 約22px + ヘッダーオフセット
- 19行目: 約403
- 20行目: 約430
- 21行目: 約457

### 5. コメント情報の読み取り

コメントが表示されると以下の情報が確認できる：
- 投稿者名
- 投稿日時
- コメント内容

スナップショットで確認する場合：
```yaml
listitem "コメントのダイアログ...":
  - generic: ナンユンティリゾーウー  # 投稿者
  - generic: 16:28 今日              # 日時
  - generic: 13時10分~               # コメント内容
```

## 報告形式

| 日付 | セル | コメント内容 | 投稿者 | 日時 |
|------|------|-------------|--------|------|
| 2日（月） | C19 | 13時10分〜 | ナンユンティリゾーウー | 16:26 今日 |

## 注意事項

- コメントマーク（セル右上の小さな三角）はスクリーンショットで見えにくい場合がある
- ユーザーにコメントがある日付を聞いてから確認するのが効率的
- セルをクリックしないとコメントは表示されない
- Shift+F2 はメモ（ノート）の編集画面を開くので、コメント確認には使わない
- ポップアップが表示された場合はEscで閉じる
