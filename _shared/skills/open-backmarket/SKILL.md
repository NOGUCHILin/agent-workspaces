# バックマーケット起動スキル

ブラウザを起動してバックマーケットの販売者管理ページ（Back Office）を開くスキル。

## 前提条件

- Playwright MCPが有効であること
- バックマーケットへのログイン情報がブラウザに保存されていること

## 手順

### 1. 販売者管理ページを開く

```
URL: https://www.backmarket.co.jp/bo-seller
```

### 2. 利用規約ダイアログを閉じる

ページ読み込み後、「利用規約」ダイアログが表示される場合がある。

- ダイアログが表示されたら「閉じる」ボタン（×）をクリックして閉じる
- ダイアログが表示されなければスキップ

## 最終状態

- バックマーケット販売者管理ページ（Back Office）が表示されている
- 利用規約ダイアログは閉じている
- タスクリストや売上インサイトが確認できる状態
