# 梱包キット催促スキル

梱包キットを発送済みだが、まだ端末を発送していない顧客を絞り込み、催促メッセージを送信するスキル。

## 前提条件

- Playwright MCPが有効であること
- kintoneにログイン済み、またはログイン情報がブラウザに保存されていること
- LINE Official Account Manager（chat.line.biz）にログイン済みであること

## ログインが必要な場合

- kintoneログイン画面 → `kintone-login` スキルを使用
- LINEログイン画面 → `open-line` スキルを使用

---

## Phase 1: kintoneで対象顧客を絞り込む

### 1. URLを開く

以下のURLを開く：

```
https://japanconsulting.cybozu.com/k/11/?view=5520280&q=f5520195%20%3C%3D%20%222026-01-24T14%3A30%3A00.000Z%22%20and%20f5520195%20%3E%3D%20%222026-01-23T15%3A00%3A00.000Z%22%20and%20f5520193%20in%20(%22%20GROUP%22%2C%20%227532782697181632512%22)%20and%20f5520277%20in%20(%225520772%22)#sort_0=f5520192&order_0=desc&size=20
```

### 2. 絞り込みボタンをクリック

「絞り込む」ボタンをクリックしてダイアログを開く。

### 3. 日付を確認・更新

絞り込みダイアログで「更新日時」の条件を確認：

- **以前（≦）**: 今日から5日前の日付、時刻は23:30
- **以降（≧）**: 今日から5日前の日付、時刻は00:00

#### 日付計算例

| 実行日 | 5日前の日付 |
|--------|-------------|
| 1/29 | 1/24 |
| 1/30 | 1/25 |
| 1/31 | 1/26 |
| 2/1 | 1/27 |

#### 日付が異なる場合

1. 「以前（≦）」の日付フィールドをクリック
2. 今日から5日前の日付を入力（YYYY-MM-DD形式）
3. 時刻は23:30のまま
4. 「以降（≧）」の日付フィールドをクリック
5. 今日から5日前の日付を入力（YYYY-MM-DD形式）
6. 時刻は00:00のまま

### 4. 適用

「適用」ボタンをクリック。

### 5. 対象顧客リストを確認

絞り込み結果から以下の情報を取得：

| 列名 | 用途 |
|------|------|
| レコード番号 | LINE検索に使用 |
| ユーザーネーム | LINE検索のフォールバック |
| ユーザー氏名 | 本人確認用 |

## Phase 2: 各顧客にLINEで催促メッセージを送信

### 各顧客について以下を繰り返す：

#### Step 1: LINE Chatでレコード番号を検索

1. LINE Chatタブに切り替え（なければ https://chat.line.biz/ を開く）
2. 左上の検索欄にレコード番号を入力
3. 「メッセージを検索」をクリック
4. 検索結果からユーザーのチャットをクリック

#### Step 2: 見つからない場合

レコード番号で見つからない場合：

1. kintoneの「ユーザーネーム」列を確認
2. そのユーザーネームでLINE検索
3. それでも見つからない場合 → **スキップ**（ブロックされている可能性）

#### Step 3: 定型文を送信

1. メッセージ入力欄の上にあるボタン群から**定型文ボタン**をクリック
2. 「梱包キット催促」テンプレートを選択
3. 「選択」ボタンをクリック
4. テキストが入力欄に挿入されることを確認（ユーザー名が自動置換される）
5. 「送信」ボタンをクリック

#### Step 4: タグを追加

1. 右パネルのタグエリアで「＋」ボタンをクリック
2. 「梱包キット催促完了2026/1」タグを選択
3. 「保存」をクリック

#### Step 5: 次の顧客へ

kintoneタブに戻り、次の顧客を処理。

## 変更しない条件

以下の条件は固定（変更不要）：

- **更新者**: Administrators
- **進捗**: 梱包キット発送完了
