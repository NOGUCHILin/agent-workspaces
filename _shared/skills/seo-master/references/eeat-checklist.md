# E-E-A-Tチェックリスト

## Experience（体験）

### 必須要素
- [ ] 実際の買取事例を掲載（各記事に最低1つ）
- [ ] 具体的な日付・金額・状態を明記
- [ ] ビフォー・アフター写真（プライバシー配慮）
- [ ] お客様の声（許可取得済み）

### 実装例
```markdown
**実例**: 2025年9月15日、横浜店にて美品のiPhone 15 Pro 128GBを76,000円で買取。
お客様からは「他社より5,000円高かった」との声をいただきました。
```

### 買取事例の要素
| 要素 | 必須 | 例 |
|------|------|-----|
| 日付 | ○ | 2025年9月15日 |
| 店舗/方法 | ○ | 横浜店/宅配 |
| 機種 | ○ | iPhone 15 Pro 128GB |
| 状態 | ○ | 美品/画面割れ |
| 価格 | ○ | 76,000円 |
| 写真 | △ | 実物写真 |

## Expertise（専門性）

### 必須要素
- [ ] 監修者情報を明記（名前、資格、経歴）
- [ ] 「AppleBuyers査定チーム監修」の表示
- [ ] 古物商許可番号の記載
- [ ] 執筆者のプロフィール

### 実装例
```markdown
---
**監修**: AppleBuyers査定チーム（古物商許可 第301031006445号、実務経験10年以上）
**執筆**: 田中太郎（iPhone買取専門ライター、5年の実務経験）
---
```

### 専門性を示す要素
- 業界固有の知識（相場変動、査定基準）
- 技術的な説明（バッテリー劣化、修理歴の影響）
- データに基づく分析（自社買取実績）

## Authoritativeness（権威性）

### 必須要素
- [ ] 信頼できる外部データの引用
- [ ] 出典URLを必ず明記
- [ ] 自社データの統計分析
- [ ] 第三者認証・受賞歴

### 実装例
```markdown
> 2025年9月の自社買取実績（128件）に基づくと、
> iPhone 15 Pro 128GBの平均買取価格は76,000円でした。
>
> 出典: [AppleBuyers買取データベース（2025年9月）]
```

### 権威性の根拠
| 種類 | 例 |
|------|-----|
| 実績数 | 15万人以上の買取実績 |
| 評価 | Google口コミ4.5/5 |
| 認証 | 古物商許可、ISO認証 |
| メディア | テレビ出演、雑誌掲載 |

## Trust（信頼性）

### 必須要素
- [ ] 更新日時の表示
- [ ] 次回更新予定の表示
- [ ] 正直な情報提供（デメリットも記載）
- [ ] 連絡先情報の明記
- [ ] プライバシーポリシー

### 実装例
```markdown
📅 **最終更新**: 2025年10月5日
🔄 **次回更新予定**: 2025年11月（相場変動を反映）

**注意**: 画面割れがある場合、買取価格は20-50%減額されます。
```

### 信頼性チェック
- [ ] デメリットも正直に記載しているか
- [ ] 誇大表現を避けているか
- [ ] 価格は最新か（更新頻度）
- [ ] 連絡先は機能しているか

## YMYL対応

### 金融情報として
- 買取価格は変動することを明記
- 「最大」「〜円」などの表現に注意
- 確定価格でないことを明示

### 記載例
```markdown
※上記は参考価格です。実際の買取価格は端末の状態・市場相場により変動します。
正確な査定額は無料査定をご利用ください。
```
