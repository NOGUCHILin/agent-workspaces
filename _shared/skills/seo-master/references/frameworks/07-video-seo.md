# 動画SEOフレームワーク

## 概要

動画SEOは、YouTube検索・Google検索・SNSでの動画発見性を高める施策。2025年、動画はSERP上位表示に大きく貢献。

## フレームワーク一覧

### 1. YouTubeランキング要因フレームワーク

**主要ランキング要因**:
| 要因 | 重要度 | 説明 |
|------|--------|------|
| 総視聴時間 | 最高 | 動画全体の累計視聴時間 |
| 視聴維持率 | 高 | 動画をどこまで見たか |
| CTR | 高 | サムネイルからのクリック率 |
| エンゲージメント | 高 | いいね、コメント、共有 |
| 登録増加率 | 中 | 視聴後のチャンネル登録 |
| セッション時間 | 中 | 視聴後も他動画を見続けるか |

**ワークフロー**:
```
1. ターゲットキーワードを選定
2. 競合動画を分析（上位10件）
3. 視聴維持を意識した構成設計
4. サムネイル・タイトル最適化
5. 説明文・タグ設定
6. 公開後のエンゲージメント促進
```

### 2. 動画メタデータ最適化フレームワーク

**タイトル最適化**:
- 60文字以内
- キーワードを前半に配置
- 数字・具体性を含める
- 疑問形・ハウツー形式が有効

**説明文最適化**:
```markdown
構造:
[最初の150文字 - 重要キーワード・要約]

[詳細説明 - 500-1000文字]

[タイムスタンプ - 主要セクション]
00:00 イントロ
01:30 ポイント1
...

[関連リンク]
[CTA - チャンネル登録、関連動画]

[ハッシュタグ - 3-5個]
#キーワード1 #キーワード2
```

**タグ設定**:
- メインキーワード（1-2語）
- ロングテールキーワード（3-5語）
- 関連キーワード
- ブランド名

### 3. サムネイル最適化フレームワーク

**効果的なサムネイル要素**:
| 要素 | 効果 | 実装 |
|------|------|------|
| 人の顔 | 高 | 表情豊かな顔を大きく |
| テキスト | 高 | 3-5語、大きく読みやすく |
| コントラスト | 高 | 明るい色、背景との差 |
| ブランド一貫性 | 中 | 色・フォント統一 |
| 好奇心喚起 | 高 | 結果をチラ見せ |

**サイズ・仕様**:
- 解像度: 1280×720px（最小）
- アスペクト比: 16:9
- ファイルサイズ: 2MB以下
- 形式: JPG, PNG, GIF

**CTR向上チェック**:
- [ ] モバイルで読める文字サイズ
- [ ] 競合と差別化されたデザイン
- [ ] タイトルと重複しない情報
- [ ] ブランドカラー・ロゴ

### 4. 視聴維持率最適化フレームワーク

**構成パターン**:
```
[フック - 最初の15秒] - 視聴継続の理由を明示
[ループオープン] - 後で回答する疑問を提示
[本編] - 価値提供、飽きさせない工夫
[CTA] - 登録、いいね、コメント促進
[エンドスクリーン] - 次の動画へ誘導
```

**離脱防止テクニック**:
- パターン中断（カット、効果音、画面切替）
- ストーリーテリング
- 視覚的変化（B-roll、図解）
- プレビュー（「この後〇〇が起きます」）

### 5. 構造化データフレームワーク（動画）

**VideoObject Schema**:
```json
{
  "@context": "https://schema.org",
  "@type": "VideoObject",
  "name": "動画タイトル",
  "description": "動画の説明",
  "thumbnailUrl": "https://example.com/thumbnail.jpg",
  "uploadDate": "2025-01-06T08:00:00+09:00",
  "duration": "PT5M30S",
  "contentUrl": "https://example.com/video.mp4",
  "embedUrl": "https://www.youtube.com/embed/VIDEO_ID",
  "interactionStatistic": {
    "@type": "InteractionCounter",
    "interactionType": "https://schema.org/WatchAction",
    "userInteractionCount": 12345
  }
}
```

**埋め込み動画のSEO**:
- 動画専用ページを作成
- 動画の上下にテキストコンテンツ
- VideoObject Schemaを実装
- 動画サイトマップ作成

### 6. YouTube Shorts最適化フレームワーク

**Shorts仕様**:
- 60秒以下
- 縦型（9:16）
- ループ視聴を意識

**最適化ポイント**:
| 要素 | 対応 |
|------|------|
| フック | 最初の1秒で注目 |
| ペース | 速いカット、間延び禁止 |
| テキスト | 字幕必須、大きく見やすく |
| CTA | 登録促進を自然に |
| ハッシュタグ | #Shorts必須 |

### 7. Google検索での動画表示フレームワーク

**動画リッチリザルト獲得**:
- VideoObject Schema実装
- 動画サイトマップ提出
- 動画専用ページ作成
- 高品質サムネイル設定

**Key Moments（重要な瞬間）対応**:
```markdown
説明文にタイムスタンプを記載:
00:00 イントロダクション
01:30 ステップ1: 準備
03:00 ステップ2: 実行
...
```

または Clip Schema を実装。
