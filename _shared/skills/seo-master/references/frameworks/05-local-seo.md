# ローカルSEOフレームワーク

## 概要

ローカルSEOは、地域検索での可視性を高め、店舗への来店やローカルサービスへの問い合わせを増やす施策。

## フレームワーク一覧

### 1. Googleビジネスプロフィール（GBP）最適化フレームワーク

ローカルSEOの最重要資産。

**完全プロフィール構成要素**:
| 要素 | 重要度 | 最適化ポイント |
|------|--------|----------------|
| ビジネス名 | 必須 | 正式名称、キーワード詰め込み禁止 |
| カテゴリ | 必須 | プライマリ+セカンダリを正確に |
| 住所 | 必須 | NAP一貫性 |
| 電話番号 | 必須 | ローカル番号推奨 |
| 営業時間 | 必須 | 祝日も含め正確に |
| 説明文 | 高 | キーワード含む、750文字活用 |
| 写真/動画 | 高 | 外観、内観、チーム、商品 |
| サービス | 高 | 提供サービスを詳細に |
| 投稿 | 中 | 週1回以上更新 |

**効果データ**:
- 完全プロフィール = 信頼性2倍
- 写真あり = 方向検索42%増、サイトクリック35%増

**ワークフロー**:
```
1. GBPを作成/オーナー確認
2. 全フィールドを100%入力
3. 高品質な写真を10枚以上追加
4. サービスと商品を登録
5. 投稿を週1回以上
6. Q&Aセクションを活用
7. レビュー対応体制を構築
```

### 2. NAP一貫性フレームワーク

Name, Address, Phoneの一貫性管理。

**チェックポイント**:
- [ ] GBPと完全一致
- [ ] 自社サイトと一致
- [ ] 各ディレクトリと一致
- [ ] SNSプロフィールと一致

**一般的な不一致例**:
```
❌ 株式会社ABC vs ABC株式会社
❌ 1-2-3 vs 1丁目2番3号
❌ 03-1234-5678 vs 0312345678
```

**ワークフロー**:
```
1. 正式なNAPを決定
2. 自社サイトに統一表記
3. GBPを更新
4. 主要ディレクトリを更新
5. 定期的に一貫性チェック
```

### 3. ローカルキーワード戦略フレームワーク

地域キーワードの最適化。

**キーワードパターン**:
| パターン | 例 |
|----------|-----|
| サービス + 地域 | iPhone買取 秋葉原 |
| 地域 + サービス | 秋葉原 スマホ買取 |
| 近くの + サービス | 近くのiPhone買取 |
| サービス + 駅名 | iPhone買取 秋葉原駅 |

**ワークフロー**:
```
1. サービスキーワードをリスト化
2. 地域修飾語をリスト化
3. 組み合わせでキーワード生成
4. 検索ボリュームを確認
5. ページごとにターゲット設定
6. 自然に本文に組み込み
```

### 4. レビュー管理フレームワーク

口コミはランキング要因かつ信頼性指標。

**レビューの重要性**:
- Googleがランキング判定に使用
- ユーザーの意思決定に直接影響
- AI Overviewにも引用される

**対応ルール**:
| 種類 | 対応 |
|------|------|
| ポジティブ | 感謝を伝える、具体的に言及 |
| ネガティブ | 迅速に、謝罪、解決策提示、オフライン誘導 |
| 偽レビュー | Googleに報告 |

**ワークフロー**:
```
1. レビュー通知を設定
2. 全レビューに48時間以内に返信
3. ポジティブレビューを促進
4. ネガティブには誠実に対応
5. 改善点をサービスに反映
6. 定期的にレビュー分析
```

### 5. ローカルリンクビルディングフレームワーク

地域に根ざした被リンク獲得。

**リンク獲得先**:
- 地域ディレクトリ
- 商工会議所
- 地域メディア
- 地域イベント協賛
- 地元企業との相互リンク

### 6. ローカルコンテンツフレームワーク

地域に特化したコンテンツ作成。

**コンテンツタイプ**:
- 店舗ページ（各店舗ごとに個別ページ）
- 地域ガイド
- 地域イベント情報
- お客様の声（地域名入り）
- スタッフ紹介
