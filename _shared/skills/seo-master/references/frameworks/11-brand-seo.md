# Brand SEO / Reputation Management

ブランド名を含む検索クエリへの自社対策ガイド（2024-2025ベストプラクティス）。

## なぜ自社対策が必要か

- **検索結果を自社でコントロール**: 口コミサイトや競合に検索結果を取られない
- **指名検索はGoogleの信頼指標**: 2024年3月コアアップデートで重要性が増加
- **ネガティブ検索への先手対応**: 「怪しい」等のクエリに透明性で応答

## 対策が必要なクエリパターン

| パターン | 検索意図 | 対策コンテンツ |
|----------|----------|----------------|
| [ブランド名] 口コミ | 利用者の評価を知りたい | 顧客レビュー・体験談 |
| [ブランド名] 評判 | 信頼できるか判断したい | 実績・メディア掲載・受賞歴 |
| [ブランド名] 怪しい | 不安を解消したい | 会社概要・資格・透明性説明 |
| [ブランド名] 詐欺 | 悪評がないか確認 | 信頼性の証明・FAQ |
| [ブランド名] vs [競合] | 比較検討中 | 比較表・差別化ポイント |

## 自社対策ページの構成

### 「口コミ・評判」ページ

```markdown
# [ブランド名]の口コミ・評判

## お客様の声

[実際のレビュー（写真・金額・日付付き）]

## SNS・メディアでの評価

[Twitter/Instagram埋め込み、メディア引用]

## よくある質問

### Q: 本当に記載通りの金額で買取してもらえますか？
A: [具体的な回答]

## まとめ
```

### 「怪しい？信頼性」ページ

```markdown
# [ブランド名]は怪しい？信頼性を徹底解説

## 会社概要
- 創業年、代表者、所在地
- 古物商許可証番号
- 取引実績（件数、金額）

## なぜ「怪しい」と検索されるのか
[業界の課題を説明し、自社の透明性をアピール]

## 信頼の証明
- メディア掲載実績
- 顧客満足度データ
- 第三者認証

## FAQ
[不安への直接回答]
```

## SEO実装のポイント

### 1. FAQスキーマ必須

```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "[ブランド名]は怪しい会社ですか？",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "..."
    }
  }]
}
```

### 2. 内部リンク戦略

- トップページから「信頼性」ページへリンク
- 全記事のフッターに「会社概要」リンク
- 価格ページから「口コミ」ページへ

### 3. メタタグ最適化

```html
<title>[ブランド名]の口コミ・評判｜実際の利用者の声を紹介</title>
<meta name="description" content="[ブランド名]の口コミ・評判を徹底解説。実際に利用したお客様の声、買取事例、よくある質問にお答えします。">
```

## 効果測定

### GSCスクリプトでの確認

```bash
# ブランドクエリの順位追跡
GSC_KEY_FILE=./key.json node scripts/track-rankings.js \
  https://example.com \
  "[ブランド名] 口コミ" \
  "[ブランド名] 評判" \
  "[ブランド名] 怪しい"
```

### KPI

| 指標 | 目標 |
|------|------|
| ブランドクエリ1位率 | 80%以上 |
| CTR | 5%以上 |
| 自社ページ表示率 | 検索結果1ページ目の50%以上 |

## 逆SEO（押し下げ）との違い

| 手法 | 目的 | 推奨度 |
|------|------|--------|
| **自社コンテンツ対策** | 良質なコンテンツで上位獲得 | ◎ 推奨 |
| 逆SEO（押し下げ） | ネガティブ記事を下位に | △ 最終手段 |
| 削除依頼 | 誹謗中傷の削除 | ○ 必要に応じて |
