# 外部SEOフレームワーク

## 概要

外部SEOは、サイト外部からの評価（被リンク、メンション、ブランド認知）を高める施策。2025年も依然として重要なランキング要因。

## フレームワーク一覧

### 1. 被リンク獲得フレームワーク

質の高い被リンクを獲得する戦略。

**2025年の被リンク評価基準**:
| 要素 | 重要度 | 説明 |
|------|--------|------|
| トピック関連性 | 高 | 同じ業界・テーマからのリンク |
| 編集基準 | 高 | 編集者が判断して設置したリンク |
| 文脈配置 | 中 | 本文中の自然な位置 |
| ユーザーエンゲージメント | 中 | 実際にクリックされるリンク |
| ドメイン権威 | 中 | 信頼性の高いサイトから |

**リンク獲得手法**:

#### ホワイトハット手法
| 手法 | 説明 | 効果 |
|------|------|------|
| Reverse Outreach | 価値あるコンテンツで自然獲得 | 最高 |
| Skyscraper | 既存上位コンテンツを超える | 高 |
| オリジナルデータ | 調査・統計・レポート公開 | 高 |
| ゲストポスト | 関連ブログへの寄稿 | 中 |
| 壊れたリンク修復 | 404リンクの代替提案 | 中 |
| リソースページ | 業界リソースへの掲載 | 中 |

#### 避けるべき手法
- 有料リンク購入
- 低品質ディレクトリ
- PBN（プライベートブログネットワーク）
- 自動化スパムアウトリーチ

**ワークフロー**:
```
1. リンク価値のあるコンテンツを作成
   - オリジナルデータ/調査
   - 包括的ガイド
   - インフォグラフィック
2. ターゲットサイトをリサーチ
3. パーソナライズしたアウトリーチ
4. 関係構築（いきなりリンク依頼しない）
5. 被リンク獲得
6. 効果測定・分析
```

### 2. ブランドSEOフレームワーク

ブランド検索と評判管理。

**構成要素**:
| 要素 | 施策 |
|------|------|
| ブランド検索 | ブランド名での検索結果を最適化 |
| ナレッジパネル | Google Knowledge Panel獲得 |
| 評判管理 | 口コミ・レビュー対応 |
| メンション | ブランド言及のモニタリング |

**ワークフロー**:
```
1. ブランド名検索結果を監視
2. 公式サイト・SNSを上位表示
3. ナレッジパネル情報を管理
4. ネガティブ情報に対応
5. ポジティブコンテンツを増やす
```

### 3. デジタルPRフレームワーク

メディア露出による権威性構築。

**施策**:
- プレスリリース配信
- メディア取材対応
- 業界イベント登壇
- 専門家コメント提供
- ケーススタディ公開

**ワークフロー**:
```
1. ニュースバリューのあるネタを準備
2. ターゲットメディアをリスト化
3. プレスリリース作成・配信
4. メディアリレーション構築
5. 掲載結果をトラッキング
```

### 4. ソーシャルシグナルフレームワーク

SNSからの間接的SEO効果。

**注意**: ソーシャルシグナルは直接のランキング要因ではないが、以下の間接効果あり:
- コンテンツ露出増加
- 被リンク獲得機会
- ブランド認知向上
- トラフィック増加

**プラットフォーム別戦略**:
| プラットフォーム | 活用方法 |
|-----------------|----------|
| X (Twitter) | 速報、業界ニュース共有 |
| LinkedIn | B2B、専門性アピール |
| YouTube | 動画コンテンツ |
| Instagram | ビジュアルコンテンツ |

### 5. E-E-A-T外部シグナルフレームワーク

外部からの信頼性シグナル。

**獲得すべき外部シグナル**:
- 業界団体への所属
- 認証・資格の取得
- 受賞歴
- メディア掲載
- 専門家としての引用

## 被リンク分析指標

| 指標 | ツール | 目安 |
|------|--------|------|
| 参照ドメイン数 | Ahrefs, Moz | 業界平均以上 |
| ドメインレーティング | Ahrefs | 40以上 |
| トラストフロー | Majestic | 30以上 |
| リンクの多様性 | 各ツール | 自然な分布 |
