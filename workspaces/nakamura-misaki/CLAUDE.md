## メンバー状況確認

各メンバーのワークスペースを確認する際は以下を参照：

```bash
# メンバー一覧
ls ../

# 各メンバーのCLAUDE.md（作業メモ含む）
cat ../{username}/CLAUDE.md

# 各メンバーのリポジトリ
ls ../{username}/repos/
```
