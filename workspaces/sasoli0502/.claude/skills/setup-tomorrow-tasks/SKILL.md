# 明日のタスク設定スキル

シフト表を参照して、Notionの作業管理DBに明日のタスクを自動作成するスキル。

## 使い方

### 基本コマンド

```bash
cd claude-workspace/.claude/skills/setup-tomorrow-tasks/scripts
node create-tasks.js --hassou "発送担当" --denwa "電話番" --mise "店担当" --ura "裏担当1,裏担当2" --date 2026-01-25
```

### オプション

| オプション | 説明 |
|-----------|------|
| `--hassou <名前>` | 発送準備担当（ネコポス担当） |
| `--denwa <名前>` | 電話番 |
| `--mise <名前>` | 店舗担当 |
| `--ura <名前>` | 裏/レ担当（カンマ区切りで複数人） |
| `--hinshitsu <名前>` | 品質管理担当（佐々木悠斗のみ） |
| `--date <日付>` | 対象日付（YYYY-MM-DD、省略時は明日） |
| `--dry-run` | 実行せずプレビュー |

### 自動機能

- **修理担当**: 雑賀晴士・高橋諒がいれば自動検出
- **名前マッピング**: シフト表の略称・旧字体を自動変換

## 実行例

### 1. シフト表を確認

```bash
node read-shift-schedule.js --date 2026-01-25
```

### 2. タスク作成

```bash
node create-tasks.js \
  --hassou "シャ" \
  --denwa "江" \
  --mise "平山優大" \
  --ura "江口那都,野口器,佐々木悠斗,高橋諒,THIRI ZAW OO" \
  --hinshitsu "佐々木悠斗" \
  --date 2026-01-25
```

### 3. 確認

```bash
node check-tasks.js 2026-01-25
```

## 仕組み（クリーンスレート方式）

1. **既存タスク全削除**: 対象日のタスクを全てアーカイブ
2. **新規作成**: テンプレートから担当者設定済みでタスク作成
3. **冪等性**: 何度実行しても同じ結果

## タスク構成（68件）

| 役割 | タスク数 | タスク例 |
|------|----------|----------|
| 発送準備 | 10件 | 梱包キット作成、発送、整理整頓など |
| 電話番 | 1件 | カスタマー対応(電話) |
| 店 | 16件 | 店頭査定、買取カスタマー対応など |
| 裏 | 7件×人数 | 査定、出品、商品撮影など |
| 修理 | 3件×人数 | 修理(販売商品)、修理(査定商品)など |
| 品質管理 | 1件 | 品質管理 |
| その他 | 2件 | 割れパネル販売、買取価格変更 |
