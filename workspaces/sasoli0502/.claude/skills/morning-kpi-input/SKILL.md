# 朝の金額KPI入力

毎朝行う金額KPIの入力作業を自動化するスキル。

## 概要

- **目的**: 前日のKPIデータを各ソースから取得し、Googleスプレッドシートの金額KPIシートに入力
- **入力先**: B列、F〜K列、R〜S列
- **データソース**: LINE Official Account Manager、Google広告、Yahoo広告

## 入力データまとめ

| 項目 | 列 | データソース | 備考 |
|------|-----|--------------|------|
| LINE配信数 | B | LINE メッセージリスト → 配信済み → 配信人数 | 前日の値。配信なしは0 |
| Google広告 成約数 | F | Google広告 概要 | コンバージョン数（整数） |
| Google広告 成約単価 | G | Google広告 概要 | コンバージョン単価（整数） |
| Google広告 費用 | H | Google広告 概要 | 費用合計（整数） |
| Yahoo広告 成約数 | I | Yahoo広告 レポート | 成約数（整数） |
| Yahoo広告 成約単価 | J | Yahoo広告 レポート | 成約単価（整数） |
| Yahoo広告 費用 | K | Yahoo広告 レポート | 費用（整数） |
| LINE友達追加 | R | LINE ホーム → 友だち | 友だち追加の累計 |
| LINEターゲットリーチ | S | LINE ホーム → 友だち | ターゲットリーチの累計 |

## 手順

### Step 1: Googleスプレッドシートを開く

```
URL: https://docs.google.com/spreadsheets/d/1Gg4Lvvlx25GGk-LdEnr8apUO2Q4e2ZOYovaAlBfV7os/edit?pli=1&gid=888185656#gid=888185656
シート: 金額KPI
```

### Step 2: 前日の日付の行番号を確認

1. `Ctrl+F` で検索ダイアログを開く
2. 日付を検索（例: `1/21`）
3. 検索結果を確認し、対象年の日付を見つける
   - 形式: `26/01/21`（2026年1月21日）
   - 複数ヒットする場合は「次へ」で正しい年を探す
4. A列のセル番号を確認（例: A1151 → 行番号は1151）

## LINE配信数の入力（B列）

### Step 3: LINE Official Account Managerにログイン

#### 3-1. ログインページを開く

```
URL: https://account.line.biz/login
```

#### 3-2. LINEアカウントでログイン

1. 「LINEアカウント」ボタンをクリック（※ビジネスアカウントではない）
2. 認証情報を入力:
   - メールアドレス: koyomiwoyomuyomu0502@gmail.com
   - パスワード: koyomi8910
3. 「ログイン」ボタンをクリック

#### 3-3. 本人確認（必要な場合）

- 認証番号が表示されたら、スマートフォンのLINEアプリで入力
- 認証完了後、ページが自動遷移する
- 400エラーが出た場合は「Go Home」をクリック

### Step 4: アップルバイヤーズを選択

LINE Official Account Managerのアカウントリストから「アップルバイヤーズ」（82,413友だち、スタンダードプラン）をクリック

### Step 5: 配信数を確認

1. 左メニューの「メッセージ配信」→「メッセージリスト」をクリック
2. 「配信済み」タブをクリック
3. 前日の日付の配信を探す
   - **配信履歴がある場合**: 「配信人数」の値を記録
   - **配信履歴がない場合**: 配信数は **0**

### Step 6: B列に配信数を入力

1. スプレッドシートの名前ボックスに `B` + 行番号（例: `B1151`）を入力してEnter
2. 配信数を入力してEnter

## Google広告データの入力（F, G, H列）

### Step 7: Google広告にアクセス

```
URL: https://ads.google.com/aw/overview?ocid=407079569&euid=592647351&__u=9861655999&uscid=407079569&__c=5657099081&authuser=0
```

### Step 8: 日付を前日に設定

1. 日付ピッカー（概要ページ上部）をクリック
2. カレンダーから前日の日付を選択（開始日・終了日両方）
3. 「適用」をクリック

### Step 9: データを確認

概要ページに表示されるデータを確認:
- **コンバージョン数**: キャンペーン別の合計を計算
- **コンバージョン単価**: ¥XX,XXX 形式で表示
- **費用**: ¥XX万 形式で表示（例: ¥13.9万 = 139,000円）

### Step 10: F, G, H列にデータを入力

1. スプレッドシートの名前ボックスに `F` + 行番号（例: `F1151`）を入力してEnter
2. 成約数（コンバージョン数）を入力してEnter
3. 名前ボックスに `G` + 行番号を入力してEnter
4. 成約単価を入力してEnter
5. 名前ボックスに `H` + 行番号を入力してEnter
6. 費用を入力してEnter

## Yahoo広告データの入力（I, J, K列）

### Step 11: Yahoo広告にログイン

```
URL: https://ads.yahoo.co.jp/
```

#### ログイン方法

1. 「Yahoo! JAPAN ID」でログイン
2. メールアドレス: info@applebuyers.jp
3. SMS認証が必要な場合、ユーザーに確認コードを依頼

### Step 12: アカウント「ecot」を選択

アカウント一覧から「株式会社ecot」を選択

### Step 13: レポートを生成

1. 右上のメニューバーから「レポート」をクリック
2. テンプレート「yahoo KPI」を選択
3. レポートを生成・ダウンロード

### Step 14: CSVからデータを抽出

ダウンロードしたCSVファイルから前日の行を探し、以下のデータを確認:
- **成約数**: 2列目
- **成約単価**: 3列目
- **費用**: 4列目

### Step 15: I, J, K列にデータを入力

1. スプレッドシートの名前ボックスに `I` + 行番号（例: `I1151`）を入力してEnter
2. 成約数を入力してEnter
3. 名前ボックスに `J` + 行番号を入力してEnter
4. 成約単価を入力してEnter（小数点以下は四捨五入）
5. 名前ボックスに `K` + 行番号を入力してEnter
6. 費用を入力してEnter

### Step 16: 小数点以下を削除（必要な場合）

1. `I:K` 列の該当行を選択
2. ツールバーの「小数点以下の桁数を減らす」ボタンを数回クリック

## LINE友達数・リーチ数の入力（R, S列）

### Step 17: LINE Official Account Managerのホームに戻る

```
URL: https://manager.line.biz/account/@906dpbsc
```

### Step 18: 友だちデータを確認

ホームページの「友だち」セクションに表示されるテーブルから前日の列を確認:
- **友だち追加**: R列に入力
- **ターゲットリーチ**: S列に入力

### Step 19: R, S列にデータを入力

1. スプレッドシートの名前ボックスに `R` + 行番号（例: `R1151`）を入力してEnter
2. 友だち追加の値を入力してEnter
3. 名前ボックスに `S` + 行番号を入力してEnter
4. ターゲットリーチの値を入力してEnter
