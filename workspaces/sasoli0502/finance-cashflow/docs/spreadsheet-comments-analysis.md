## 技術的な注意点

### CSV vs xlsx形式

| 形式 | コメント | 書式 | 数式 |
|------|---------|------|------|
| CSV | ❌ 失われる | ❌ 失われる | ❌ 計算結果のみ |
| xlsx | ✅ 保持される | ✅ 保持される | ✅ 保持される |

### コメント読み取り方法

```python
import openpyxl

wb = openpyxl.load_workbook('財務管理表.xlsx')
ws = wb['資金繰り表']

for row in ws.iter_rows():
    for cell in row:
        if cell.comment:
            print(f'{cell.coordinate}: {cell.comment.text}')
```
