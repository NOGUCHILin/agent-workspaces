# Decision Rules - 判断ルール

請求書カード払いの判断基準とルール

---

## 請求書カード払いサービス一覧

3つのサービスを使い分ける。**手数料は全て3%（×1.03）**

| サービス | 期日超過 | 利用可能カード | 備考 |
|----------|---------|---------------|------|
| **マネーフォワード** | ❌ 不可 | Visa/Master/JCB（全カード対応） | **最優先**（手数料最安水準） |
| **請求書カード払い byGMO** | ❌ 不可 | **アメックス専用** | アメックス利用時のみ |
| **DGFT** | ✅ **可能** | Visa/Master/JCB（**トヨタ✗、GPC✗**） | 期日超過時のみ使用 |

### サービス選択フロー

```
期日を超過して支払う？
├─ YES → DGFT（※トヨタ・GPC使用不可）
└─ NO → カードは何？
         ├─ アメックス → byGMO
         └─ それ以外 → マネーフォワード
```

## カード優先順位（引落日が遅い順）

### 基本方針
**引落日が最も遅くなるカードを優先する**

### 月後半（16日〜月末）に振込日を設定する場合

| 優先順位 | カード | 締め日 | 引落日 | 猶予目安 |
|----------|--------|--------|--------|----------|
| 1位 | **しんきんVisa/JCB** | 15日 | 翌月10日 | 最大56日 |
| 2位 | **アメックス** | 22日 | 翌月10日 | 最大49日 |
| 3位 | Amazon | 月末 | 翌月26日 | 最大56日 |
| 4位 | トヨタ | 5日 | 翌月2日 | 変動 |

※ 決済日が締め日を超えると翌月分になり、引落がさらに1ヶ月遅れる

## 請求書分割ルール

### 分割可能
**徐さん（仕入）のみ**
- 請求書を分割してもらうことが可能
- 例：150万円 → 100万円 + 50万円の2枚の請求書に分割
- 目的：複数カードに分けて支払い、枠を効率的に使用

### 分割不可
**ヤマト、ビズビ、家賃3件（ユニホー、ドンキ、レンタルオフィス）**
- 請求書分割不可
- 1枚の請求書 = 1つのカードで全額支払い必須

## 申請期限の計算方法

- **申請期限**: 振込希望日の**4営業日前**まで
- **決済期限**: 振込希望日の**2営業日前**まで
- **営業日判定**: `holidays.csv` を参照（土日祝日を除く）

## 期日超過戦略（ユニホー・ビズビ向け）

期日を超過して支払うことで、引落日を大幅に遅らせる戦略。

### 例：12月末期日の請求書

**通常（期日厳守）**
- 振込12/31 → 決済12/26 → Amazon: 引落2026/1/26

**期日超過戦略**
- 申請1/1 → 振込1/5 → 決済1/2 → Amazon: 引落**2026/2/26**（1ヶ月遅延）

### 注意点
- DGFT経由でのみ可能（マネーフォワード・byGMOは期日超過不可）
- DGFTでは**トヨタカード・GPCは使用不可**

---

## 判断フロー

```
1. 期日を確認
   ├─ 厳守必須（ヤマト・ドンキ）→ 2へ
   └─ 超過OK（ユニホー・ビズビ・徐さん）→ 3へ

2. 期日厳守の場合
   ├─ アメックス使用 → byGMO
   └─ それ以外 → マネーフォワード
   → 最も引落が遅いカードを選択

3. 期日超過する場合
   ├─ 期日超過で引落を遅らせたい → DGFT（トヨタ・GPC以外）
   └─ 期日内で払う → 2へ
```
