# 💳 クレジットカード運用最適化システム

クレジットカードの支払猶予期間を最大化するための運用支援ツール

## 🎯 主な機能

### 📊 ダッシュボード（dashboard.html）
- **カード残高入力**: 毎日の残高を手入力で更新
- **今日使うべきカード**: 引き落とし日が一番遅いカードを自動表示
- **締日アラート**: 一週間以内の締日を通知
- **請求書カード払いスケジュール**: 申請日と引き落とし日の管理
- **引き落としスケジュール**: 資金繰り管理

### 🐍 Python最適化スクリプト（scripts/）
- 月次の支払い計画を自動最適化
- カード配分の推奨
- Markdownレポート出力

## 📁 ファイル構成

```
finance-cashflow/
├── dashboard.html          # メインダッシュボード（ブラウザで開く）
├── docs/
│   └── master-data-v2.yaml # カード情報マスターデータ
├── scripts/
│   └── optimize_credit_schedule.py  # 月次最適化スクリプト
├── data/
│   └── input_202511.yaml   # 月次入力データ
└── README.md               # このファイル
```

## 🚀 使い方

### ダッシュボード（毎日）

1. `dashboard.html` をブラウザで開く
2. 各カードの利用可能額を入力
3. 「残高を反映して再計算」ボタンをクリック
4. 今日使うべきカードを確認

### Python最適化（月次）

```bash
cd finance-cashflow
uv run python scripts/optimize_credit_schedule.py data/input_202511.yaml
```
