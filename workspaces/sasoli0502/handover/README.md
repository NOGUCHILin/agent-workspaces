## 📁 ディレクトリ構成ルール

### `/work/handover/` - 他者への引継ぎ用
野口凜など他のメンバーへの引継ぎ事項（長期的に保存する情報）

### `/work/handover/personal/` - 個人用セッション引継ぎ
野口器（自分）が次のClaude Codeセッションで文脈を引き継ぐための文書
- プロジェクト進捗の記録

- 作業中断時の状況まとめ
