## ハイブリッド戦略とは

### 目的
短期的に**粗利を底上げ**しつつ、**成約率も維持**する2段階の買取価格提示システム。

### 戦略の核心

**機種×ランク**の組み合わせで初回提示価格を決定します。

| 機種分類 | ランク | 戦略 | 初回提示 | 交渉方針 |
|---------|-------|------|---------|---------|
| **人気機種** | 使用感あり・目立つ傷あり | **高粗利** | 低い買取価格 | 段階的にUP（最大2回） |
| **人気機種** | 新品同様・美品 | **安全** | 高い買取価格 | 交渉なし |
| **通常機種** | 使用感あり・目立つ傷あり | **高粗利** | 低い買取価格 | 段階的にUP（最大2回） |
| **通常機種** | 新品同様・美品 | **安全** | 高い買取価格 | 交渉なし |

## ランク別戦略

端末のランク（状態）によって、初回提示価格が変わります。

### 高粗利戦略の対象ランク
- **使用感あり**
- **目立つ傷あり**

→ これらのランクは**低い買取価格から提示**し、お客様の反応を見て段階的に価格を上げます。

### 安全戦略の対象ランク
- **新品同様**
- **美品**

→ これらのランクは**高い買取価格から提示**し、交渉の余地を残しません（成約率重視）。

## 高粗利戦略の交渉フロー

**対象**: 使用感あり・目立つ傷あり

### ステップ1: 初回提示（低い買取価格）

査定システムで算出された「**初回提示価格**」をお客様に提示します。

**例**: iPhone 15 Pro Max 1TB 使用感あり
```
初回提示価格: 105,600円
```

**伝え方**:
```
「こちらの端末の査定結果は、105,600円となります。
 いかがでしょうか?」
```

**注意**: すべての買取価格は**100円単位で切り上げ**されています。

---

### ステップ2: 断られた場合（1回目交渉）

「**1回目交渉価格**」まで引き上げます。

**例**:
```
1回目交渉価格: 109,300円
```

**伝え方**:
```
「少々お待ちください。上司に確認いたします。
 ...
 最大で109,300円までお出しできます。こちらでいかがでしょうか?」
```

---

### ステップ3: さらに断られた場合（2回目交渉）

「**2回目交渉価格**」まで引き上げます。

**例**:
```
2回目交渉価格: 110,600円
```

**伝え方**:
```
「大変申し訳ございません。もう一度上司に相談させてください。
 ...
 特別に110,600円まで対応させていただきます。」
```

---

### ステップ4: それでも断られた場合（最終ライン）

**最終ライン価格**を確認します。

**例**:
```
最終ライン価格: 110,600円（2回目交渉と同額）
```

**判断**:
- 2回目交渉価格 = 最終ライン価格 → **これ以上は引き上げられない**
- それ以上の価格は**赤字リスク**があるため、お断りする

**伝え方**:
```
「誠に申し訳ございませんが、これ以上のお値段は難しい状況です。
 ご検討いただけますと幸いです。」
```

この後、お客様が返送を希望された場合は、**[返送交渉マニュアル](返送交渉マニュアル.md)** に従って対応してください。

---

### 交渉回数の制限

- **最大2回まで**（初回提示 → 1回目 → 2回目）
- 3回以上の交渉は行わない（時間コスト削減）

## 安全戦略の交渉フロー

**対象**: 新品同様・美品

### ステップ1: 初回提示（高い買取価格）

査定システムで算出された「**初回提示価格**」をお客様に提示します。

**例**: iPhone 12 128GB 美品
```
初回提示価格: 28,600円
最終ライン価格: 28,600円（同額）
```

**伝え方**:
```
「こちらの端末の査定結果は、28,600円となります。
 こちらが当店の最高買取価格となります。」
```

**注意**: すべての買取価格は**100円単位で切り上げ**されています。

---

### ステップ2: 断られた場合

**初回提示価格 = 最終ライン価格**のため、**交渉の余地はありません**。

**伝え方**:
```
「大変申し訳ございませんが、こちらが精一杯のお値段となります。
 他社様とも比較いただき、ご検討くださいませ。」
```

この後、お客様が返送を希望された場合は、**[返送交渉マニュアル](返送交渉マニュアル.md)** に従って対応してください。

---

## 返送交渉との違い

### ハイブリッド戦略（このマニュアル）
- **タイミング**: 査定時の初回価格提示
- **目的**: 高粗利と成約率の両立
- **手法**: 機種・ランクに応じた段階的価格提示（システム算出価格を使用）

### 返送交渉（[返送交渉マニュアル](返送交渉マニュアル.md)）
- **タイミング**: 査定後にお客様が返送希望した場合
- **目的**: 返送を避けて買取成立させる
- **手法**: お客様の文脈を読んで柔軟に価格UP（スタッフの裁量あり）

**重要**: ハイブリッド戦略の最終ライン価格を提示しても断られた場合は、返送交渉マニュアルに切り替えてください。

## 実例サンプル

### 実例1: iPhone 15 Pro Max 1TB 使用感あり（高粗利戦略）

**販売価格**: 130,000円（仮定）
**人気度**: 人気
**ランク**: 使用感あり → **高粗利戦略**

| 段階 | 買取価格 | 粗利 | 利率 |
|------|---------|------|------|
| 現在価格（参考） | 110,000円 | 5,700円 | 4.9% |
| **初回提示** | **105,600円** | **10,100円** | **8.7%** |
| 1回目交渉 | 109,300円 | 6,400円 | 5.5% |
| 2回目交渉 | 110,600円 | 5,100円 | 4.4% |
| 最終ライン | 110,600円 | 5,100円 | 4.4% |

**基準**: 粗利14,000円以上 / 利率10~14%

**戦略**:
1. 初回は105,600円で提示（高粗利狙い）
2. 断られたら109,300円まで引き上げ
3. さらに断られたら110,600円まで引き上げ
4. それでも断られたら返送交渉へ移行
