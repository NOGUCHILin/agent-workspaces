## 📊 分析概要

**分析日**: 2024-11-24
**対象データ**: [買取価格20251119.csv](買取価格20251119.csv)
**分析対象**: 高額買取価格と特急買取価格の差額パターン

---

## 🎯 発見した規則性

### ✅ 基本ルール：販売価格帯に応じた差額設定

**高額買取価格と特急買取価格の差額は、高額買取価格（≒販売価格帯）に比例して決定されている**

### 📈 予測式（通常品）

```
差額 = 高額買取価格 × 0.067451 + 956.68
```

- **決定係数（R²）**: 0.8506（85%の精度で予測可能）
- **統計的有意性**: p値 < 0.000001（極めて有意）

## 💡 実務への応用

### 新しい買取価格を設定する際の手順

1. **高額買取価格を決定**（市場調査、競合価格、粗利基準）
2. **予測式で差額を計算**:
   ```
   差額 = 高額買取価格 × 0.067 + 950
   ```
3. **キリの良い金額に丸める**:
   - 1,000円未満 → 100円単位
   - 1,000円以上 → 1,000円単位
4. **特急買取価格を算出**:
   ```
   特急買取価格 = 高額買取価格 - 差額
   ```

### 計算例

**高額買取価格 75,000円の場合**:
1. 予測差額 = 75,000 × 0.067 + 950 = 5,975円
2. キリの良い金額に丸める → 6,000円
3. 特急買取価格 = 75,000 - 6,000 = 69,000円

## 📌 まとめ

1. **高額買取価格と特急買取価格の差額は、販売価格帯に比例して決定されている**
2. **差額 = 高額買取価格 × 6.7% + 950円** の数式で85%の精度で予測可能
3. **実務では1,000円単位または500円単位でキリの良い金額に丸められている**
4. **価格帯が上がるほど差額率は下がる**（低価格帯19% → 超高価格帯2-3%）
5. **主要な差額は2,000円、4,000円、6,000円、10,000円**
