# 販売価格マッピングルール

**作成日**: 2025-11-20
**対象**: 買取価格設定担当者

---

## 概要

買取価格を決定する際、マックマーケット（Back Market）の販売価格をもとに設定します。

買取価格の「ランク」と販売価格の「グレード」は異なる体系のため、以下のマッピングルールに従って販売価格を算出します。

---

## マッピングルール

### 買取価格ランク → 販売価格グレードの対応

| 買取価格ランク | 販売価格グレード | 計算方法 |
|--------------|----------------|---------|
| **新品同様** | A | グレードAの価格をそのまま使用 |
| **美品** | A + B | (グレードA + グレードB) ÷ 2 |
| **使用感あり** | B + C | (グレードB + グレードC) ÷ 2 |
| **目立つ傷あり** | C | グレードCの価格をそのまま使用 |

---

## 実例: iPhone 15 Pro Max 1TB

### 販売価格データ（マックマーケット）

```
グレードA: 149,298円
グレードB: 147,974円
グレードC: 131,132円
```

### 買取価格設定用の販売価格

```
新品同様:     149,298円 (= グレードA)
美品:         148,636円 (= (149,298 + 147,974) ÷ 2)
使用感あり:   139,553円 (= (147,974 + 131,132) ÷ 2)
目立つ傷あり: 131,132円 (= グレードC)
```

### これらの販売価格に設定された利率を適用して買取価格を決定

例: 使用感ありの買取価格を決定する場合
```
販売価格: 139,553円
手数料引後: 139,553 × 0.89 = 124,202円
目標利率: 11.0%（例）

買取価格 = 124,202 × (100 - 11.0) / 100 = 110,600円
```

---

## 注意事項

1. **販売価格データの更新頻度**: マックマーケットの価格は定期的に変動するため、週次で更新すること
2. **グレードが存在しない場合**: 一部の機種で特定のグレードが存在しない場合、近いグレードで代用するか、個別に判断すること
3. **新品・未開封**: 「新品・未開封」は別途「プレミアム」グレードを使用する場合があるため、個別確認が必要
