# 新品未開封（打痕・汚れあり）減額確認マニュアル

最終更新: 2025年11月6日

---

## 概要

新品未開封商品で外箱に打痕や汚れがある場合、買取一丁目にLINEで連絡して減額金額を確認するフローを説明します。

**重要**: 新品未開封商品の買取には**購入証明書が必須**です。商品到着前にLINEでPDFデータを受け取るか、コピーを同封して発送してもらう必要があります。

---

## 対象商品

- **新品未開封**の商品
- 外箱に以下のような状態がある場合:
  - 打痕（へこみ）
  - 汚れ
  - 傷
  - その他の外観上の問題

---

## フロー全体

```
0. 購入証明書の確認（商品到着前）
   ↓
1. 商品状態の確認
   ↓
2. 写真撮影
   ↓
3. 買取一丁目にLINE連絡
   ↓
4. 減額金額の回答を待つ
   ↓
5. 減額金額を反映して査定続行
```

---

## 詳細手順

### 0. 購入証明書の確認（商品到着前）【必須】

新品未開封商品の買取には購入証明書が必須です。商品到着前に以下のいずれかの方法で入手してください。

#### **方法1: LINEでPDFデータを受け取る（推奨）**

**お客様への依頼文例:**
```
新品未開封商品の買取には購入証明書が必要となります。
お手数ですが、購入時のレシート・領収書・購入明細書の
PDFデータまたは写真をLINEで送付いただけますでしょうか。

以下のいずれかをご提出ください:
- 店舗レシート・領収書の写真
- ECサイトの購入明細書（スクリーンショット可）
- 納品書の写真

※個人情報（住所・電話番号等）は黒塗りしていただいて構いません
※商品名・購入日・購入金額が確認できれば問題ございません
```

**確認事項:**
- [ ] 商品名が記載されているか
- [ ] 購入日が記載されているか
- [ ] 購入金額が記載されているか（任意だが望ましい）
- [ ] 画像が鮮明で読み取れるか

#### **方法2: 商品に同封して発送してもらう**

**お客様への依頼文例:**
```
新品未開封商品の買取には購入証明書が必要となります。
お手数ですが、商品発送時に以下のいずれかのコピーを
同封していただけますでしょうか。

同封していただくもの（いずれか）:
- 店舗レシート・領収書のコピー
- ECサイトの購入明細書の印刷
- 納品書のコピー

※個人情報（住所・電話番号等）は黒塗りしていただいて構いません
※商品名・購入日・購入金額が確認できれば問題ございません
```

#### **購入証明書が提出されない場合**

購入証明書の提出がない場合、以下の対応を行います:

1. **再度お客様に依頼**
   - 丁寧にLINEで再依頼
   - 提出方法を具体的に説明

2. **それでも提出されない場合**
   - **野口器または佐々木悠斗に報告**
   - 買取不可となる可能性が高い
   - お客様への説明方法を相談

**重要:**
- 購入証明書がない商品は原則として査定を進めない
- 商品到着前に必ず確認すること
- 判断に迷ったら野口器に相談

---

### 1. 商品状態の確認

新品未開封商品の査定時に、以下を確認:

- [ ] **購入証明書が提出されているか**（最優先確認事項）
- [ ] 商品は未開封状態か
- [ ] 外箱に打痕・汚れがあるか
- [ ] 打痕・汚れの程度（軽度/中度/重度）

**購入証明書がない場合:**
- 査定を一時停止
- お客様に購入証明書の提出を依頼
- 野口器または佐々木悠斗に報告

### 2. 写真撮影

**撮影内容:**
- 打痕・汚れがある箇所を明確に撮影
- 複数箇所ある場合は、すべて撮影
- 商品全体がわかる写真も1枚撮影

**撮影のポイント:**
- 明るい場所で撮影
- ピントを合わせる
- 打痕・汚れの大きさがわかるように

### 3. 買取一丁目にLINE連絡

**送信先:**
- 買取一丁目のLINEアカウント
- **アクセス方法**: 買取一丁目の公式サイト左上にある**電話マーク**を押すとLINEアカウントが表示される

**メッセージ内容:**

```
お世話になっております。
新品未開封商品の査定で確認したいことがあります。

商品: [商品名・型番]
状態: 新品未開封
問題点: 外箱に[打痕/汚れ]があります

写真を送付いたしますので、
減額金額をご教示いただけますでしょうか。

よろしくお願いいたします。
```

**写真を送信:**
- 撮影した写真をすべて送信

### 4. 減額金額の回答を待つ

**対応:**
- 回答が来るまで、その商品の査定は**保留**
- 他の商品の査定を進める
- 回答時間の目安: 通常30分～2時間程度

**回答例:**
- 「-500円で対応可能です」
- 「-1,000円の減額となります」
- 「この状態では買取不可です」

### 5. 減額金額を反映して査定続行

**買取可能な場合:**
1. 回答された減額金額をメモ
2. 通常の査定価格から減額金額を引く
3. 査定を続行

**買取不可の場合:**
1. **野口器または佐々木悠斗に報告**
2. 他の業者への確認を検討
   - 詳細な対応は野口器または佐々木悠斗が指示

---

## 注意事項

### 必ず確認が必要なケース
- 打痕が**深い**（1cm以上のへこみ）
- 汚れが**目立つ**（拭いても取れない）
- 複数箇所に問題がある

### 確認不要なケース
- 軽微な擦り傷程度
- 目立たない小さな汚れ
  - **小さな汚れの場合**: アルコールで軽く拭いてから判断
  - 汚れが取れれば通常の「新品未開封」として査定
  - 汚れが取れない場合は買取一丁目に確認

→ 汚れが取れた場合は通常の「新品未開封」として査定

**判断に迷ったら:**
- 保守的に買取一丁目に確認する
- 写真を撮って野口器に相談

## よくある質問

**Q: 購入証明書は必ず必要ですか？**
A: はい、新品未開封商品の買取には必須です。提出がない場合は原則として買取できません。

**Q: 購入証明書として認められる書類は？**
A: 以下のいずれかで問題ありません:
- 店舗レシート・領収書
- ECサイトの購入明細書・注文履歴
- 納品書
- 商品名・購入日が確認できる書類

**Q: 購入証明書の個人情報はどう扱う？**
A: お客様に「住所・電話番号等は黒塗りしていただいて構いません」と伝えてください。商品名・購入日・購入金額が確認できれば十分です。

**Q: 購入証明書がどうしても用意できないとお客様が言う場合は？**
A: 野口器または佐々木悠斗に相談してください。基本的には買取不可となります。

**Q: どの程度の打痕・汚れから確認が必要？**
A: お客様が見て「気になる」レベルなら確認する。判断に迷ったら確認を優先。

**Q: 回答が遅い場合は？**
A: 2時間以上返信がない場合は、野口器に相談。

**Q: 減額金額が大きすぎる場合は？**
A: 野口器に相談。お客様への説明方法を確認。

**Q: 複数商品を同時に確認できる？**
A: 可能。1つのメッセージで複数商品の写真を送ってOK。
