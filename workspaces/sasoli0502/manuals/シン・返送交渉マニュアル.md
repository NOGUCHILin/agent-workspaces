## このマニュアルについて

### 従来の返送交渉との違い

**従来**:
- スタッフの裁量で価格を決定
- 明確な基準がなく、判断が難しい

**シン・返送交渉**:
- **文脈を読んで柔軟に価格UP**（従来の強み）
- **明確な数値基準を提供**（新しい強み）
- 最終ライン価格を超えない範囲で対応

### 基本方針

1. **お客様の文脈を読む**（重要度：★★★）
   - 柔らかい口調 → 少し上げる
   - 「少し上がりませんか」→ 買取価格に応じて調整
   - 強い要求 → しっかり上げる

2. **明確な基準を使う**（重要度：★★★）
   - 機種・容量・ランクから価格基準を確認
   - 初回提示価格、1回目交渉価格、2回目交渉価格、最終ライン価格を把握
   - **最終ライン価格を超えない**

3. **スピード最優先**（重要度：★★★）
   - 当日中に1回目のメッセージを必ず送る
   - 可能であれば即座に送る

## 全体フロー

```
お客様「返送してください」
  ↓
スプレッドシートで確認（セル青色に）
  ↓
【機種・容量・ランクを確認】
  ↓
価格基準シートで以下を確認:
- 現高額買取価格（出発点）
- 1回目交渉価格（約1/3 UP）
- 2回目交渉価格（約2/3 UP）
- 最終ライン価格（粗利下限）
  ↓
価格を上げられる？
  ├─ YES → 交渉開始
  └─ NO  → 交渉不成立（そのまま返送手続き）
  ↓
【1回目】
文脈を読んで価格決定
・柔らかい口調 → 少し上げる（現買取価格 → 1回目交渉価格の間）
・「少し上がりませんか」→ 1回目交渉価格まで
・強い要求 → しっかり上げる（2回目交渉価格付近）
・重要: 上げ切れる上限（最終ライン価格）まで上げ切らない
・理由付けは柔軟に（選択肢が多いほど説得力UP）
  - 不具合の減額取り消し
  - 相場変動
  - 社内協議の結果
  ↓
テンプレートで送信（当日中）
  ↓
お客様の反応は？
  ├─ OK → 交渉成立
  ├─ もう少し → 【2回目】へ
  └─ 無理です → 希望金額を聞く＋返送手続き（並行）
  ↓
【2回目以降】
理由付けの柔軟さを強化:
・「上司に相談したところ…」
・感情的な表現（お客様のご希望に添えず申し訳ございません、等）
・価格を再提示（最終ライン価格に近づける）
  ↓
お客様の反応は？
  ├─ OK → 交渉成立
  └─ 無理です → 希望金額を聞く＋返送手続き（並行）
  ↓
返送手続き中に希望金額が来た
  ↓
対応可能？（最終ライン価格以下か？）
  ├─ YES → 返送キャンセル → 交渉成立
  └─ NO  → そのまま返送
```

## 粗利・粗利率の基準値

返送交渉時の**最終ライン価格**は、以下の粗利下限基準に基づいて設定されています。

### 買取価格帯別の粗利下限

| 買取価格帯 | 粗利下限 |
|-----------|---------|
| 〜2万円 | 4,000円 |
| 2〜4万円 | 6,000円 |
| 4〜6万円 | 6,000円 |
| 6〜8万円 | 8,000円 |
| 8〜10万円 | 8,500円 |
| 10〜15万円 | 9,500円 |
| 15万円〜 | 10,000円 |

**重要**:
- 最終ライン価格を超えて買取価格を引き上げると、上記の粗利下限を下回り赤字リスクが発生します
- 交渉時は必ず最終ライン価格を超えないようにしてください
- すべての買取価格は**100円単位で切り上げ**されています

## 交渉フロー詳細

### 【1回目】文脈を読んで価格を決定

#### パターン1: 柔らかい口調・軽い不満

**お客様の反応例**:
```
「もう少し高くなりませんか？」
「他社と比較したいので、少し考えます」
```

**対応**:
- **現高額買取価格** → **1回目交渉価格の中間**くらいまで引き上げ
- 理由付け: 「再度確認させていただいたところ」「相場変動」

**例**:
```
現高額買取価格: 110,000円
1回目交渉価格: 111,600円
→ 提示する価格: 110,800円〜111,300円
```

#### パターン2: 「少し上がりませんか」系

**お客様の反応例**:
```
「あと少しだけ上げてもらえると嬉しいです」
「5,000円くらい上がりませんか？」
```

**対応**:
- **1回目交渉価格**まで引き上げ
- 理由付け: 「社内協議の結果」「上司に相談したところ」

**例**:
```
現高額買取価格: 110,000円
1回目交渉価格: 111,600円
→ 提示する価格: 111,600円
```

#### パターン3: 強い要求・不満

**お客様の反応例**:
```
「他社ではもっと高い金額を提示されました」
「この金額では納得できません」
「返送でお願いします」
```

**対応**:
- **2回目交渉価格**まで引き上げ（ただし1回目では最終ラインまで出さない）
- 理由付け: 「上司に再度相談」「不具合の減額取り消し」

**例**:
```
現高額買取価格: 110,000円
2回目交渉価格: 113,200円
→ 提示する価格: 112,000円〜113,000円（最終ラインは残す）
```

---

### 【2回目以降】最終調整

1回目の交渉でも断られた場合、2回目で**最終ライン価格**に近づけます。

**対応**:
- **最終ライン価格**まで引き上げ
- 理由付けを強化: 「上司に再度相談いたしまして」「特別に対応」
- 感情的な表現を加える

**例**:
```
現高額買取価格: 110,000円
最終ライン価格: 114,800円
→ 提示する価格: 114,800円
```

**それでも断られた場合**:
- 希望金額を聞く
- 返送手続きを並行して進める
- 希望金額が最終ライン価格以下なら対応可能

## 価格の決め方（文脈を読む）

### 重要ポイント

1. **パターン判定は厳密ではない**
   - お客様の文脈次第で柔軟に対応
   - 例: 少し不満そう → ちょっとだけ価格UP
   - 例: 強い要求 → 大きく価格UP + 理由付け

2. **理由付けの選択肢**
   - 不具合の減額取り消し（カメラ・バッテリー・画面）
   - 相場変動
   - 社内協議の結果
   - 上司との相談の結果
   - 改めて確認させていただいたところ

3. **スピード最優先**
   - 当日中に1回目のメッセージを必ず送る
   - 可能であれば即座に送る（チャットを開いた人が即座にメッセージを送る）

4. **価格の上げ方**
   - 最大値（最終ライン価格）まで一気に上げない（段階的に）
   - ただし極端な値上げは避ける（不信感につながる）
   - 例: ❌ 1万円 → 5万円（お客様が疑問に思う）

5. **希望金額を聞くタイミング**
   - 2回目で断られたら
   - 返送手続きを並行して進める
   - 対応可能な金額（最終ライン価格以下）なら返送をキャンセル

6. **スプレッドシート管理**
   - 交渉中のレコードはセル色を青色に
   - 詳細はスプレッドシートを参照

## よくある質問

**Q: どのくらい価格を上げればいい？**
A: お客様の文脈次第。明確な基準はないが、価格段階（初回提示→1回目→2回目→最終ライン）を参考に。

**Q: 不具合の減額取り消しは必ずやる？**
A: お客様が納得すればどんな理由でもOK。柔軟に判断。

**Q: 外装ランクは変更できる？**
A: できない。信頼性の問題につながる。

**Q: 交渉は何往復まで？**
A: 基本的に2回程度。それ以降は希望金額を聞いて返送手続き。

**Q: 返送手続き中に希望金額が来たら？**
A: 最終ライン価格以下なら返送キャンセル。ただし手続き完了後は無理。

**Q: 最終ライン価格を超えてもいい？**
A: **絶対にダメ**。赤字リスクがあるため、最終ライン価格が上限。

**Q: 安全戦略の機種でも交渉できる？**
A: 初回提示 = 最終ラインのため、交渉の余地はほとんどない。
