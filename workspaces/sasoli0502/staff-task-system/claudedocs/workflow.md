# 1日の業務フロー

スタッフタスク管理システムの標準的な1日の業務フローです。

---

## 📅 タイムライン

```
09:30 - 朝礼・集計準備
10:00 - 朝の集計入力 & タスク自動割り振り
10:30 - 午前業務開始
13:00 - 午後の集計入力（発送関連）
14:00 - チェックポイント（午前進捗確認）
16:00 - 午後の集計入力（梱包キット）
17:00 - チェックポイント（終業確認）
18:00 - 日報生成
```

---

## 🌅 朝の業務（09:30-10:30）

### 1. 朝礼・準備（09:30）
- 今日の出勤スタッフ確認
- 特記事項の共有（欠勤、休憩変更等）

### 2. 朝の集計入力（10:00）

**Claude Codeへの指示例**:
```
"朝の集計を入力して。査定待ち50台、開封待ち30台"
```

**実行されるコマンド**:
```bash
uv run python scripts/input_morning_summary.py --satei 50 --kaifuu 30
```

**自動計算される項目**:
- 修理必要: 査定の50%
- 検品必要: 査定の50%
- 出品可能: 査定の80%
- アクティベート: 開封と同数

### 3. タスク自動割り振り（10:10）

**Claude Codeへの指示例**:
```
"今日の役割分担を決めて。出勤は細谷、江口、シャシャ、佐々木、雜賀"
```

**実行されるコマンド**:
```bash
uv run python scripts/suggest_assignments.py \
  --staff "細谷,江口,シャシャ,佐々木,雜賀" \
  --auto-create
```

**結果**:
- 全スタッフにタスクが自動割り当て
- スキルマッチング済み
- 負荷分散済み
- 10秒で完了

---

## 💡 Tips

### Claude Codeとの対話のコツ

1. **キーワード優先**
   - 「朝の集計」「役割分担」などのキーワードを使う
   - 自動でコマンド実行される

2. **ニックネーム使用可**
   - 「細谷」でも「たかひろ」でもOK
   - 「シャシャ」が「NANT」に自動変換

3. **進捗確認は随時**
   - 「今日のタスクは？」「進捗を見せて」
   - いつでも状況確認可能
