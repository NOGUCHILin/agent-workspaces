## 📝 インシデント作成ガイド

### 新規インシデントを作成する場合

1. [template.md](template.md)をコピー
2. `open/`ディレクトリに配置（ファイル名: `XXX-title.md`）
3. このINDEX.mdのOpenセクションに追加

### インシデント解決時

1. ファイルを`open/`から`resolved/`に移動
2. INDEX.mdのResolvedセクションに追加
3. Openセクションから削除

### Claude Codeとの対話例

```
# 未解決インシデントを確認
"未解決のインシデントは？"
"openディレクトリを見て"

# 新規インシデント作成
"新しいインシデントを作成して。パフォーマンス問題について"

# インシデント解決
"001のインシデントを解決済みにして"
```
