---

# 実行フロー

```
1. LINE Official Account Managerにアクセス
     ↓
2. ログイン画面が表示された場合 → LINEアカウントでログイン
     ↓
3. チャット画面の表示を確認
     ↓
4. 通知バーが表示されたら閉じる
     ↓
5. 完了報告
```

# Step 1: LINE Official Account Managerにアクセス

```
mcp__playwright__browser_navigate で https://chat.line.biz/ にアクセス
```

## ケースA: チャット画面が直接表示された場合
- 「LINE Chat」のタイトルが確認できれば完了

## ケースB: ログイン画面が表示された場合
- 「LINEヤフーBusiness ID」のページが表示される
- Step 2へ進む

# Step 2: ログイン

## 2-1. LINEアカウントを選択

```
「LINEアカウント」ボタンをクリック
```

## 2-2. アカウント選択画面

ログイン済みセッションがある場合:
- 「えぐち なつ」のアカウントが表示される
- 「ログイン」ボタンをクリック

ログイン済みセッションがない場合:
- メールアドレスとパスワードの入力が必要
- ユーザーにログイン操作を依頼する

## 2-3. 認証後の遷移を待機

- ログインボタンクリック後、タイムアウトエラーが出ても遷移は進んでいる場合がある
- `mcp__playwright__browser_snapshot` でページ状態を確認
- URL が `chat.line.biz` になり、タイトルが「LINE Chat」であればOK

# Step 3: 通知バーを閉じる

チャット画面表示後、通知バー（alert要素）が表示される場合がある:

```
「ブラウザ通知は[チャット設定]＞[基本設定]＞[通知方法]から設定できます。」
```

- 「閉じる」ボタンが表示されていればクリックして閉じる
- 表示されていなければスキップ

# Step 4: 完了報告

「LINE Chatを開きました」と報告して終了。

# 認証情報

| 項目 | 値 |
|------|-----|
| サービス | LINE Official Account Manager |
| URL | https://chat.line.biz/ |
| ログイン方法 | LINEアカウント |
| アカウント名 | えぐち なつ |
| 公式アカウント名 | アップルバイヤーズ（Pro） |

# 注意事項

- ログインセッションが切れている場合は、LINEアカウントで再ログインする
- セッションが完全に切れてメール・パスワード入力が必要な場合はユーザーに操作を依頼する
