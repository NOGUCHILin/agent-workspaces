# Multi-Agent Research System

出典: [Anthropic Engineering](https://www.anthropic.com/engineering/multi-agent-research-system)

---

## 概要

ClaudeのResearch機能のための本番マルチエージェントシステム。
並列エージェント協調による複雑な情報収集を実現。

---

## アーキテクチャ: Orchestrator-Worker Pattern

```
Lead Agent（Opus 4）
  ├── 分析・戦略策定
  │
  ├── Subagent 1（Sonnet 4）── 並列探索
  ├── Subagent 2（Sonnet 4）── 並列探索
  ├── Subagent 3（Sonnet 4）── 並列探索
  │
  └── Citation Agent ── ソース帰属確保
```

---

## パフォーマンス

> 「Opus 4リード + Sonnet 4サブエージェントは、単一Opus 4を**90.2%**上回る」

---

## 主要な学び

| 項目 | 説明 |
|------|------|
| **トークン経済** | 単一ターンの約15倍、タスク価値が正当化する場合に有効 |
| **プロンプト設計** | 厳格なルールより研究ヒューリスティクスをエンコード |
| **評価戦略** | 小規模テストセット（20クエリ）で劇的改善を確認 |

---

## 本番課題

- 状態が多ターンにまたがり、失敗が予測不能にカスケード
- 非決定論的動作がデバッグを複雑化
- 同期サブエージェント実行がボトルネック
- Rainbowデプロイで更新時の中断を防止
