# ワークフロー

1. 目的をヒアリング（話す音声 or 歌声 or BGM）
2. 適切なツール選択
3. テキスト/歌詞の準備
4. 生成 → 必要に応じて調整

# ツール選択

| 目的 | おすすめツール | 特徴 |
|------|---------------|------|
| 話す音声（ナレーション） | ElevenLabs | 高品質、自然な感情表現、日本語対応 |
| 歌声・オリジナル曲 | Suno | 歌詞から作曲・編曲・ボーカル自動生成 |
| 無料・キャラクター音声 | VOICEVOX | 完全無料、商用OK、個性的な声 |
| BGM・効果音 | Suno / Udio | 歌詞なしの音楽生成 |

# 各ツール詳細

## ElevenLabs（話す音声）

- **用途**: ナレーション、ポッドキャスト、動画吹き替え
- **日本語**: 対応済み（自然なイントネーション）
- **アクセス**: elevenlabs.io

### Tips
- 句読点で自然な間を作る
- 感情指定: 「excited」「calm」「serious」
- 声のクローンも可能（自分の声を学習）

## Suno（歌声・音楽）

- **用途**: オリジナル楽曲、BGM、SNSコンテンツ
- **日本語**: 対応済み
- **アクセス**: suno.com

### 日本語で良い結果を得るコツ
- 歌詞をひらがな/カタカナにすると発音が安定
- メタタグで構造指定: `[Verse]` `[Chorus]` `[Bridge]`
- ジャンル指定: J-pop, anime style, city pop

### プロンプト例
```
[Verse]
きみとあるいた あのみちを
おもいだしてる なつのひ

[Chorus]
わすれないよ このきもち
ずっとずっと きえないから

Style: J-pop, emotional ballad, female vocal
```

## VOICEVOX（無料・キャラクター）

- **用途**: 動画ナレーション、ゲーム、商用利用
- **特徴**: 完全無料、個性的なキャラクターボイス多数
- **アクセス**: voicevox.hiroshiba.jp（デスクトップアプリ）
